# Sample Output Files

This directory contains realistic simulated output files from quantum chemistry codes, provided for teaching purposes and to demonstrate how to parse computational chemistry results programmatically.

## Files

| File | Code | Method | Molecule | Calculation Type |
|------|------|--------|----------|----------------|
| `water_opt_orca.out` | ORCA 6.1 | B3LYP/def2-SVP | H₂O | Geometry optimization |
| `fe_co5_freq_orca.out` | ORCA 6.1 | BP86/def2-SVP | Fe(CO)₅ | Frequency calculation |

> **Note:** These output files are simulated for educational use. They reproduce the key sections and formatting of real ORCA output but were not generated by running ORCA directly.

---

## How to Parse with cclib

[cclib](https://cclib.github.io) is a Python library that parses output files from many quantum chemistry codes (ORCA, Gaussian, Jaguar, Psi4, etc.) into a uniform data structure.

```python
import cclib

# Parse an ORCA output file
data = cclib.io.ccread("water_opt_orca.out")

# Inspect available attributes
print(dir(data))

# Access final geometry (in Angstroms)
print("Optimized coordinates (Angstrom):")
print(data.atomcoords[-1])       # shape: (natoms, 3)
print("Atomic numbers:", data.atomnos)

# SCF energies across optimization steps (in eV)
print("SCF energies (eV):", data.scfenergies)

# For frequency calculations
freq_data = cclib.io.ccread("fe_co5_freq_orca.out")
print("Vibrational frequencies (cm-1):", freq_data.vibfreqs)
print("IR intensities (km/mol):", freq_data.vibirs)
```

### Useful cclib attributes

| Attribute | Description |
|-----------|-------------|
| `data.atomcoords` | Cartesian coordinates for all steps, shape `(nsteps, natoms, 3)` Å |
| `data.atomnos` | Atomic numbers, shape `(natoms,)` |
| `data.scfenergies` | SCF energies in eV for each optimization step |
| `data.vibfreqs` | Vibrational frequencies in cm⁻¹ (frequency jobs) |
| `data.vibirs` | IR intensities in km/mol |
| `data.moenergies` | Molecular orbital energies in eV |
| `data.homos` | Index of the HOMO for each spin channel |
| `data.charge` | Molecular charge |
| `data.mult` | Spin multiplicity |

## How to Generate Your Own ORCA Outputs

### Geometry Optimization (water)

```
# water_opt.inp
! B3LYP def2-SVP Opt TightSCF

* xyz 0 1
O   0.000000   0.000000   0.117176
H   0.000000   0.757001  -0.468704
H   0.000000  -0.757001  -0.468704
*
```

Run with:
```bash
orca water_opt.inp > water_opt.out
```

### Frequency Calculation (Fe(CO)5)

```
# fe_co5_freq.inp
! BP86 def2-SVP Freq TightSCF

* xyzfile 0 1 fe_co5.xyz
```

Run with:
```bash
orca fe_co5_freq.inp > fe_co5_freq.out
```

ORCA is available free of charge for academic use from the [ORCA forum](https://orcaforum.kofo.mpg.de).
