# Images and Figures

This directory stores static images and figures used in the Jupyter Book.

## Contents

Place book figures here, e.g.:
- Scheme diagrams (reaction schemes, molecular orbital diagrams)
- Pre-rendered plots referenced in notebooks
- Logo files

## Naming Convention

Use descriptive, lowercase filenames with hyphens:

```
ch01-basis-set-convergence.png
ch08-crystal-field-splitting.png
ch10-tddft-uv-spectrum.png
```

Prefer **PNG** for diagrams/screenshots and **SVG** for vector graphics.
Reference images in notebooks with standard Markdown:

```markdown
![Crystal field splitting diagram](../assets/images/ch08-crystal-field-splitting.png)
```
